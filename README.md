# Codeforces Analytics Dashboard

A full-stack web application that visualizes Codeforces performance using interactive charts.

## Features
- JWT based authentication
- Protected dashboard
- Codeforces API integration
- Rating history (line chart)
- Rating-wise problem distribution (bar chart)
- Tag-wise problem distribution (pie chart)
- Server-side caching with manual refresh
- Dark mode UI

## Tech Stack
- Frontend: React, Tailwind CSS, Recharts
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT
